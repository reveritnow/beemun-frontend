# BEEMUN Frontend

BEEMUN is a curated ecommerce marketplace for ZPS 100 products.

## Brand Promise

**Pure for You. Pure for Earth.**

Every product on BEEMUN should be reviewed around three trust pillars:

- **Zero Plastic**
- **Zero Synthetic**
- **Full Disclosure**
- **BEEMUN Reviewed**

## Frontend Stack

Planned stack:

- Next.js
- React
- Tailwind CSS
- shadcn/ui
- Vercel deployment

## Build Direction

The UI should feel:

- premium
- natural
- calm
- earthy
- trustworthy
- curated
- transparent

It should not feel like a generic Amazon/Flipkart-style marketplace.

## Development Rule

Do not commit secrets to this repository.

Use environment variables for API URLs, payment keys, upload keys, and any private configuration.
